# User Stories

## 1. Autenticacao e Cadastro
- US-001: Como usuario, quero me cadastrar com nome, email e senha para acessar a loja.
- US-002: Como usuario, quero fazer login com email e senha para acessar minha conta.
- US-003: Como usuario, quero recuperar minha senha por email caso eu esqueca.

## 2. Gerenciamento de Produtos
- US-004: Como usuario, quero visualizar a lista de produtos disponiveis.
- US-005: Como usuario, quero visualizar os detalhes de um produto especifico.
- US-006: Como administrador, quero adicionar novos produtos ao catalogo.
- US-007: Como administrador, quero editar um produto existente.
- US-008: Como administrador, quero excluir um produto do catalogo.

## 3. Carrinho de Compras
- US-009: Como usuario, quero adicionar produtos ao meu carrinho.
- US-010: Como usuario, quero visualizar os itens do meu carrinho.
- US-011: Como usuario, quero atualizar a quantidade de um item no carrinho.
- US-012: Como usuario, quero remover um item do carrinho.

## 4. Compras e Historico
- US-013: Como usuario, quero finalizar a compra dos itens no carrinho.
- US-014: Como usuario, quero visualizar meu historico de compras.
- US-015: Como usuario, quero visualizar o detalhe de uma compra.

## 5. Interface do Usuario
- US-016: Como usuario, quero ver a lista de produtos na interface da loja.
- US-017: Como usuario, quero ver uma pagina de carrinho com totais.
- US-018: Como usuario, quero revisar o pedido na pagina de checkout.

## 6. Administracao
- US-019: Como administrador, quero gerenciar produtos em um painel administrativo.
- US-020: Como administrador, quero visualizar o historico de compras dos usuarios.
